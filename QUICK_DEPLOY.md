# ⚡ Deploy Rápido com Mock Data (5 minutos)

## 3 Passos Simples:

### 1️⃣ Preparar o Código
```bash
# Copia o ficheiro de configuração
cp .env.local.example .env.local

# Commit
git add .env.local.example
git commit -m "chore: add deployment example"
git push
```

### 2️⃣ Fazer Deploy no Vercel
1. Vai a https://vercel.com/new
2. Seleciona o teu repositório GitHub
3. Clica "Deploy" (sem adicionar environment variables!)
4. Pronto! ✅

### 3️⃣ Partilhar o Link
Copia o link que Vercel te dá e envia:
```
🌐 https://seu-dashboard.vercel.app
```
