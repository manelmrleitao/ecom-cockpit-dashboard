# 🚀 Ecom Cockpit Dashboard

Dashboard centralizado que agrega dados de múltiplas plataformas de publicidade (Google Ads, Meta Ads, TikTok Ads) e conecta com Shopify para análise de performance em tempo real.

## 📋 Stack

- **Frontend**: Next.js 16+ (App Router) + React 19 + TypeScript
- **Backend**: Next.js API Routes + Node.js
- **Database**: Supabase (PostgreSQL)
- **Deploy**: Vercel
- **Styling**: Tailwind CSS 4
- **Charts**: Recharts
- **Data Fetching**: TanStack Query
- **Validation**: Zod

## 🚀 Quick Start

### 1. Configurar Variáveis de Ambiente

```bash
cp .env.example .env.local
```

### 2. Instalar Dependências

```bash
npm install
```

### 3. Correr em Desenvolvimento

```bash
npm run dev
```

Abra [http://localhost:3000/dashboard](http://localhost:3000/dashboard) no navegador.
